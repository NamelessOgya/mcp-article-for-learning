# get_paper_citations

## 概要
指定された論文タイトルやキーワードを使用してSemantic Scholarを検索し、その論文を**引用している他の論文（被引用論文）**の情報を取得します。
取得されるデータは、被引用数（citationCount）が多い順にソートされて返されます。

## 機能
- Semantic Scholarの Graph API を使用して論文を検索します。
- 最初に見つかった論文を対象とし、それを引用している文献を取得します（1回のリクエストで最大1000件）。
- 取得した文献リストを、**さらに他の論文から引用されている数（影響力の高さ）** の降順でソートします。
- ユーザーが指定した上限件数（`limit`）に従い、上位の論文の「タイトル」「Abstract」「被引用数」を文字列として整形して返します。
- **無償APIの利用制限を回避するため、検索リクエストと引用取得リクエストの間に1秒の待機（Wait）処理が入ります。（実行に数秒かかる場合があります）**

## 使い方 (LLMプロンプトの例)
- 「"Attention is all you need" を引用している影響力のある論文を5件教えて」
- 「"BERT" の論文を引用している最新の研究は？ (limit=10で引いて内容を確認)」

## 入力パラメータ
| パラメータ名 | 型 | 必須 | デフォルト値 | 説明 |
| :--- | :--- | :--- | :--- | :--- |
| `query` | `str` | Yes | - | 検索対象の論文タイトルまたはキーワード（完全一致推奨） |
| `limit` | `int` | No | 10 | 取得して返す引用論文の最大件数 |

## 戻り値
検索結果として、元の論文タイトルと、それを引用している上位論文のリスト（タイトル、被引用数、Abstractを含むテキスト）を返却します。
エラー時は、エラーメッセージの文字列を返却します。
